운영진 예제 스킬 실행
운영진이 준비한 Subagent 예제 하나 돌려보기. 결과 품질보다 "병렬 호출이 실제로 됐는가"를 로그에서 본다.
1
$ git clone <운영진 repo>/recap-mail-parallel skills/
예제 스킬 로컬 복제.
2
$ cat skills/recap-mail-parallel/SKILL.md
상단에 "Subagent 2개 동시 호출" 선언 확인.
3
(you) /skill recap-mail-parallel 고객 X
실행 — 메인이 Subagent 2개를 병렬로 띄운다.
4
(Claude) Launching 2 subagents in parallel...
Subagent 로그 확인 — 시간 스탬프가 겹치면 병렬 성공.
결과 내용은 덤. 오늘 관찰 포인트는 "병렬이 보이는가"만.
