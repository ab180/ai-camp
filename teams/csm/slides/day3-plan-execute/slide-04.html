오늘 구조
블록 5개 · 외부 시선 + 반복 적용 + 3패턴 + 협업 한 사이클
오늘 120분은 "굴러보기"가 중심. 어제 PR을 짝꿍이 점검하고, 본인 케이스로 사이클을 반복하고, Subagent 3패턴을 익힌 뒤 PR을 머지까지 한 바퀴 돌린다.
B0
~15 min
어제 스킬 짝꿍 점검
어제 만든 스킬을 짝꿍이 한 번 호출 + 빈틈 메모
B1
~40 min
사이클 반복 적용
본인 케이스 1·2·(3) — 70 → 90% 체감이 핵심
B2
~35 min
Subagent 본격
3패턴(병렬·역할 분리·컨텍스트 보호) + 역할 분리 실습
B3
~20 min
Git/PR 협업 한 사이클
어제 PR 받기 → 수정 → 머지 + pull까지 한 바퀴
B4
~10 min
회고 + Day 4 예고
1줄 회고 → 내일 첫 검증 케이스
산출물 5종 = 짝꿍 점검 메모 + 사이클 결과 2-3건 + Subagent 1회 + PR 머지+pull + 회고 1줄.
