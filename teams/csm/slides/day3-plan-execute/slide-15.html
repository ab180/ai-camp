Deep Interview 스킬 호출 순서
처음이라면 새 Claude Code 세션 진입. 기존 세션에서도 OK. 중요한 건 5~8개 질문에 끝까지 답하기.
1
$ claude
Claude Code 세션 진입 — 새 폴더에서도 OK. 기존 프로젝트 폴더면 더 좋음.
2
(you) 나 고객 X 리캡 메일 스킬을 만들고 싶어
문제 1줄 입력. "만들고 싶어"라는 의도를 명확히 말하기.
3
(you) /skill deep-interview 시작해줘
Deep Interview 스킬 실행 — 자동 질문이 시작됨.
4
(Claude) 이 스킬의 독자는 누구인가요? ...
5~8개 질문에 답. 모르는 건 "모른다"고 답해도 OK — 기본값 제안됨.
첫 시도는 대답이 짧아도 OK. 목표는 완벽한 답이 아니라 스펙 문서 1장 받아내기.
