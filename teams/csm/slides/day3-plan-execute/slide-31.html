새 PR 1개(선택) — git log 3개가 변화량
시간이 남으면 새 브랜치로 PR 한 번 더. 핵심은 머지 후 본인 git log 3개 commit 누적 — 어제 → 오늘 변화량 시각화.
$ git checkout -b day3-cycle-pattern  # 새 브랜치
$ vi members/<이름>.md  # Day 3 패턴 1줄 추가
$ git add . && git commit -m "feat: Day 3 사이클 패턴"
$ git push -u origin day3-cycle-pattern
# GitHub 웹에서 PR 생성 → 짝꿍 리뷰어 지정 → 머지
머지 후 git log — 3일 누적
