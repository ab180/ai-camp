## B2 · Subagent 본격 (9장) · S15–S24

### S15 · 섹션 오프너
- Block 2 · ~40min
- 제목: Subagent 본격 — 3패턴 + CSM 실습

### S16 · Subagent 개념
- 한 줄 정의: 메인 AI가 독립 AI를 일회성 호출 — 결과만 받는다
- 3특성: 독립 컨텍스트 / 병렬 처리 / 결과만 반환

### S17 · 3패턴 개요
- 패턴 1 병렬 검색 → 시간 단축
- 패턴 2 역할 분리 → 품질 향상 (오늘 실습)
- 패턴 3 컨텍스트 보호 → 컨텍스트 절약

### S18 · 패턴 1 병렬 검색 상세
- before/after: 순차 150초 vs 병렬 30초
- CSM 예: 미팅 5건 동시 액션 추출

### S19 · 패턴 2 역할 분리 상세
- Generator(작성) → Evaluator(검토) 다이어그램
- 핵심: 만든 사람은 자기 맹점을 못 본다

### S20 · 패턴 3 컨텍스트 보호 상세
- before/after: 100페이지 직접 vs Subagent+핵심 3줄
- 3패턴 요약 셀

### S21 · CSM 역할 분리 실습 · ~15분
- 3단계: Generator 결과 준비 / Evaluator 검토 요청 / 메모
- 관찰 포인트: gap이 역할 분리의 가치

### S22 · CSM 사용처
- 3케이스: 미팅 리캡 마감(병렬) / 갱신 리스크 메일(역할) / 체크업 자료(컨텍스트)

### S23 · Subagent 주의점 3가지
- 맥락 공유 X / 비용 증폭 / 결과 합치기 책임

### S24 · 짝꿍 활용 대화 · 5분
- Q1: 순차로 하고 있는데 병렬로 돌리면 좋을 것
- Q2: 3곳 이상에서 정보 모아야 하는 작업
- Q3: 끝나길 기다리면서 딴 일 하는 패턴
