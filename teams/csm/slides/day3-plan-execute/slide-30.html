main에서 pull

~2분
$ git checkout main && git pull  # 5단계: 머지된 변경 가져오기
$ git log --oneline -5  # 본인 commit이 main에 들어왔는지 확인
push가 이미 끝나도 머지 전엔 내 변경이 main에 없다. 머지 → pull까지 가야 "main에 내가 만든 게 있다"는 체감이 온다.
