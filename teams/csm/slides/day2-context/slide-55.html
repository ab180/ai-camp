Git ≠ GitHub, 다른 것이다
비슷하게 들려도 전혀 다른 두 가지. 헷갈리면 대화가 꼬인다. 어디에서 돌아가는지로 구분하면 쉽다.
Git

내 컴퓨터에서 돌아가는 버전 관리 프로그램

터미널에서 실행하는 프로그램.
  • 내 컴퓨터 안에서만 작동
  • 로컬 파일 변경을 스냅샷으로 추적
  • 터미널 명령어로 조작 (`git ...`)
  • 인터넷 없어도 동작
GitHub

Git으로 관리한 파일을 올려두는 인터넷 공간

브라우저에서 접속하는 클라우드 서비스.
  • 웹 주소가 있는 서버 (github.com)
  • 팀원이 볼 수 있도록 파일 저장·공유
  • PR·이슈·리뷰 같은 협업 기능
  • 인터넷 필수
Git은 내 컴퓨터에서 돌아가고, GitHub은 클라우드 — 다른 것. 둘 다 필요.
