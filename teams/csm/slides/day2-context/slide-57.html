개념 지도 · 04
4개 풀어쓰기 — 각 단어가 무엇인지
한 장 안에 Git 명령어(작게 · 참고용)와 Claude에게 줄 자연어(크게 · 메인)를 함께. 후자가 실제로 쓰는 방식.

clone

01 · 가져오기
GitHub에 있는 집을 내 컴퓨터로 복사해서 가져오기.
Git
git clone <url>
Claude
"이 레포 clone해줘"

branch

02 · 평행우주
원본은 그대로 두고 내 작업용 평행우주 만들기.
Git
git checkout -b my-feature
Claude
"새 브랜치 파줘"

commit

03 · 한 줄 저장
여기까지 한 일을 한 줄로 적고 저장. 나중에 이 시점으로 돌아올 수 있음.
Git
git commit -m "CLAUDE.md 톤 추가"
Claude
"커밋해줘"

PR

04 · 허락 받기
내 평행우주를 원본에 합쳐달라고 쪽지 보내기. 리뷰 받고 머지.
Git
GitHub 웹에서 버튼
Claude
"PR 올려줘"
