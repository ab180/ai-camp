Troubleshooting
이 3가지 중 하나면 바로 해결된다
MCP 연결 실패 패턴의 90%는 아래 3가지 중 하나. 증상을 보고 바로 찾아 적용 가능하도록 정리.
증상
/mcp가 서버 없음이라 응답
원인
Claude Code 세션이 연결 전에 시작됐음
해결
세션 끄고 다시 실행. 30초 안에 해결.
증상
Connect 눌러도 OAuth 창이 안 뜸
원인
브라우저 팝업 차단 · 광고 차단 확장
해결
주소창 옆 팝업 아이콘 → 허용 → 재시도
증상
권한 승인 후에도 scope 0
원인
Google Workspace 관리자 정책 제한
해결
운영진 공유 · 계정 관리자 승인 요청
15분 안에 안 되면 다른 도구로 전환. 오늘 산출물은 "1개 연결"이지 특정 도구 아님 — 시도했다는 것 자체가 학습.
