GitHub 웹에서 — 내 첫 Repo 만들기
터미널 없이 GitHub 페이지에서 클릭 4번으로 repo를 만든다. 이 repo가 오늘 저녁 과제·내일·모레 스킬을 담을 집이 된다.
1
GitHub → "+" → New repository
오른쪽 위 + 버튼 클릭 → 드롭다운에서 "New repository" 선택.
2
Repository name 입력
이름은 ai-camp-practice-<본인이름>. 소문자·하이픈만 (공백·한글 X).
3
Private + README 체크
Private 선택 (처음엔 안전). 그리고 "Add a README file" 체크박스 ON.
4
Create repository
하단 녹색 Create repository 버튼 한 번. 끝.
만들면 자동으로 main 브랜치가 생긴다. 이 repo에 이따 내 CLAUDE.md 한 섹션을 올린다 — 다음 슬라이드부터 그 흐름.
