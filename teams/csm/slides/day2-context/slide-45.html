최소 세트
이 5개면 오늘은 충분하다
Git 명령어는 수십 개지만 — CSM 입장에서 오늘 이 5개만. 나머지(branch · rebase · stash)는 2주 프로젝트에서 자연스레.
git clone
남의 repo를 내 로컬에 복제
공용 스킬을 처음 받을 때. Day 1에 이미 한 번 했음.
git pull
최신 상태로 갱신
팀이 수정한 걸 내 로컬에 반영할 때.
git commit
변경사항을 "스냅샷"으로 박제
오늘 내가 한 작업에 이름을 붙여 저장. 되돌릴 수 있는 단위.
git push
내 커밋을 Github에 올림
팀이 볼 수 있게 공개. 내 로컬에만 있던 게 원격에 복사.
PR (Pull Request)
"내 변경을 팀 repo에 합쳐줘"
Github 웹에서 생성 · 리뷰 후 merge.
오늘 밤 과제는 5개 전부 써봄 — 한 번 흐름을 타면 다음부턴 익숙. 바로 다음 슬라이드부터 구체 흐름.
