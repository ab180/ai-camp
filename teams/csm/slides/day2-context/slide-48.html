STEP 1 / 2
흐름 · 방금 만든 내 repo
내 repo에 첫 커밋 올리는 5단계
앞 슬라이드에서 만든 내 repo를 기준으로 움직인다. 처음 해보면 헷갈리지만 한 번만 흐름을 타면 다음부턴 익숙해진다.
1
git clone <내 repo URL>
Github Code 버튼 → URL 복사 후 터미널에서 clone. 내 로컬에 폴더 생성됨.
2
폴더 구조 확인 + CLAUDE.md 섹션 추가
cd로 들어가 README 등 파일 구조 눈으로 확인. 에디터에서 CLAUDE.md 열어 담당 고객 섹션 한 줄 추가.
3
git commit -m "feat: 담당 고객 섹션 추가"
수정을 스냅샷으로 박제. 메시지에 `<type>: <설명>` 형식 — feat · fix · docs 중.
4
git push
내 로컬 커밋을 Github에 업로드. 웹에서 새로고침하면 변경사항이 보임.
5
(심화) 팀 repo에서 PR 생성
내 repo로 흐름 익힌 뒤 팀 repo에 같은 흐름 반복. PR 생성은 다음 슬라이드에서.
3번 커밋 메시지는 <type>: <설명> 형식. feat(새 기능) · fix(수정) · docs(문서) · refactor(구조 개선) 중.
