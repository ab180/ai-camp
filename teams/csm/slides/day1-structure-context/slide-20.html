실습 1
스킬 받아오기
4단계를 순서대로 따라간다. 명령어를 이해하려 하지 말고 일단 돌려보고 뒤에 관찰한다. 터미널은 운영진이 사전 준비한 로그인·권한 상태.
01
git clone <repo-url>
운영진이 준비한 공용 repo를 내 로컬로 복제. Slack에 공유된 URL을 그대로 붙여넣는다.
02
cd <repo>
클론된 폴더로 이동. Claude Code는 현재 폴더 기준으로 스킬·CLAUDE.md를 읽는다.
03
claude
Claude Code 세션 시작. 터미널 안에서 대화 인터페이스가 열린다. 여기부터가 "AI와 일하는 자리".
04
"리캡 메일 초안 써줘 — 고객사: 샘플 A"
스킬 호출. 이 한 줄이
공용 스킬 + 샘플 A 데이터를 묶어 초안을 만든다.
