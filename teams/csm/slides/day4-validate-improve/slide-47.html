부록 2 · Supabase
팀 공용 DB + API 5분
팀 공유는 GitHub로 끝나지 않는다. 실제 데이터 (케이스 · 신호 · 로그)를 함께 쌓으려면 DB가 필요. Supabase는 코드 없이 UI에서 만든다.
Supabase = PostgreSQL DB + REST API + 인증을 5분 안에. UI에서 코드 한 줄 없이 테이블 생성 가능. 개인 프로젝트 무료.
왜 CSM에게 — 3가지 시나리오
시나리오 1

케이스 DB

지난 문의 · 답변을 팀 단위로 누적. 비슷한 케이스 검색 가능.

시나리오 2

wrap 신호 누적

본인 wrap 신호를 팀 차원으로. 팀 단위 Compound 사이클.

시나리오 3

스킬 실행 로그

어떤 스킬이 자주 쓰이나 데이터로. 우선순위 결정 근거.

안 쓰는 경우 — 1인 작업만 / 외부 클라우드 보안팀 미승인 / 고객 PII 저장 (별도 DPA 필요).
