팀 PR은 만들었다. 외부 사람이 권한 없이 보게 하려면? Vercel은 GitHub repo를 자동으로 공개 URL로 배포해주는 무료 서비스. 코드 한 줄도 안 건드린다.
Vercel = GitHub repo를 자동으로 공개 URL로 배포하는 서비스. push할 때마다 자동 재배포 · 무료 Hobby 플랜으로 충분.
