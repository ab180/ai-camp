# Recap Mail — CSM 미팅 리캡 메일 초안

> axes: [3, 4]  | owner: @yechan | last-update: 2026-04-23

## when
고객 미팅 후 리캡 메일 초안이 필요할 때.
특히 결정사항 · 책임자 · 기한을 명확히 써야 하는 경우.

## input
  - 미팅 노트 (Notion 또는 Slack 스레드)
  - 참석자 리스트
  - 고객 관계 단계 (신규 / 운영 / 갱신)

## output
  - 제목 (25자 이내)
  - 결정사항 3개 이하 · 한 문장씩
  - 다음 액션 (담당자 @mention + 기한)
  - 문의 · 블로커

## how to use
  1. /recap-mail 실행
  2. 미팅 노트 붙여넣기
  3. 고객 관계 단계 입력
  4. 까다로운 고객 에이전트에 검수 → 수정

## example
실제 리캡 메일 1건 링크 (민감 정보 제거)
필드가 많지 않다. when · input · output 3개만 있어도 팀원이 쓸 수 있다. 나머지는 시간 나면 채운다.
