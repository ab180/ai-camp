수동 사이클 → 자동 트리거
Compound 사이클을 사람이 매주 따로 시간 잡아 돌리려면 잘 안 된다. wrap이 매 세션 끝에 자동으로 Notice를 적고, 임계치 도달 시 알림을 주면 인지 부담이 사라진다.
기존 — 수동

매주 회고 시간을 따로 잡아야

  • 1.매주 회고 시간 따로 — 일정 잡기 어렵다
  • 2.사람이 직접 Notice 적기 — 잘 안 적힌다
  • 3.패턴 발견은 우연 — 3회 반복 알아채기 어렵다
신규 — wrap 자동

매 세션 끝에 자동으로

  • 1.세션마다 자동 — 따로 시간 안 잡아도 된다
  • 2.wrap이 Notice 자동 작성 — 누락 X
  • 3.임계치 도달 시 알림 — 3회 반복 자동 감지
