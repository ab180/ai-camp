업로드 실습 · 15 min
clone → 복사 → commit → push
오늘 각자 스킬 1개를 팀 repo에 올린다. Claude에게 아래 5단계를 순서대로 시키면 된다.
1
git clone
팀 repo를 내 노트북에 가져온다 — “csm-claude-skills 레포를 clone해줘”
2
cd & branch
새 브랜치로 이동 — “add-recap-mail 브랜치 만들어서 이동해줘”
3
폴더 복사
Day 3에서 만든 내 스킬 폴더를 skills/에 복사. SKILL.md 6축 태그 확인.
4
commit
“recap mail 스킬 추가” 메시지로 commit — 세이브 포인트 1개 생성
5
push
GitHub에 올린다 — “origin add-recap-mail으로 push해줘”
명령어는 Claude가 실행. 참가자는 의도만 말하면 된다. 막히면 짝꿍과 화면 공유 · 운영진 호출.
