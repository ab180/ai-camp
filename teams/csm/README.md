# AI Camp · CSM 팀

AB180 CS Group 전용 AI Camp 자산.

## 시작하기

1. `claude` 명령으로 Claude Code 세션 시작
2. Day별 슬라이드는 `slides/dayN-<theme>/viewer.html` 로 열람
3. 템플릿 스킬은 `templates/`
4. 개인 스킬은 `.claude/skills/` 에 추가

## 구조

```
teams/csm/
├── slides/              Day별 발표 슬라이드
├── templates/           운영진 사전 제작 스킬 템플릿 (0.5→1 수정 대상)
├── .claude/skills/      내/팀 스킬 작업 공간
└── guides/              발표자 가이드
```

## 참고
